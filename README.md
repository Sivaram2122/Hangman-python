# Python Hangman Game

This is my very first Python project! It is a classic, text-based Hangman game where the player tries to guess a hidden word one letter at a time before running out of attempts.
## 🎮 Features
- Randomly selects a secret word from a built-in list.
- Tracks guessed letters to prevent the player from guessing the same letter twice.
- Displays a visual representation of remaining attempts (like a stick figure drawing).
- Validates user input to ensure players only enter valid letters.
## ⚙️ Installation & Setup

To play this game on your local machine, make sure you have **Python 3** installed.

1. Clone or download this repository to your computer.
2. Ensure both `hangman.py` and your wordlist file (e.g., wordlist.py) are placed inside the exact same folder.
## 🚀 How to Run the Game

1. Open your terminal or command prompt.
2. Navigate to the folder where you saved the files.
3. Run the following command:

```bash
python hangman.py
```
